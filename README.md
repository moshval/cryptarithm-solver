# Deskripsi Singkat
Dapat menyelesaikan persoalan penjumlahan cryptarithmetic\
Input berupa file dengan ekstensi .txt, sebagai contoh: (contoh lain terdapat pada folder test)
```
 SEND
 MORE+
------
MONEY
```
